# Support

## 使用问题

先运行：

```bash
npm run doctor -- --ready
npm run project:status -- <slug> --compact-json
```

然后搜索已有 [Issues](https://github.com/cyberlesterr/paper-collage-video/issues)。如果仍未解决，请提交 Bug report，并附上：

- 操作系统、Node.js、Python 和 FFmpeg 版本；
- 插件版本或 commit；
- 执行的命令和完整错误；
- 去除密钥、私人提示词和未授权素材后的最小复现项目。

## 支持边界

社区支持按维护者可用时间提供，不承诺响应时效或兼容所有第三方 provider。以下内容不应通过普通 Issue 处理：

- 安全漏洞：使用 [SECURITY.md](SECURITY.md) 的私密渠道；
- API key、付费账号或 provider 账单问题：联系相应服务商；
- 素材、声音、品牌或肖像授权判断：由内容所有者自行确认；
- Remotion 商业许可证：查看 [Remotion License](https://github.com/remotion-dev/remotion/blob/main/LICENSE.md)。
